Installation
---------------------------------

dcapy depends on the following libraries:
* NumPy
* Pandas
* Statsmodels

Matplotlib is optionally required, if one uses the built-in plotting

dcapy can be installed via pip::

   pip install dcapy

or from source::

   git clone https://github.com/matt-black/dcapy
   cd dcapy
   python setup.py
